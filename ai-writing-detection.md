# AI Writing Detection Reference

Words, phrases, and patterns that reliably mark text as AI-generated. Every item below is a **hard prohibition**. None of these may appear in a finished post.

*Sources: Grammarly (2025), Microsoft 365 Life Hacks (2025), GPTHuman (2025), Walter Writes (2025), Textero (2025), Plagiarism Today (2025), Rolling Stone (2025), MDPI Blog (2025)*

---

## Em Dashes: The Primary AI Tell

**The em dash (—) has become one of the most reliable markers of AI-generated content.**

AI models drastically overuse em dashes because they were trained on edited books, academic papers, and style guides where em dashes appear frequently. AI uses them as a shortcut for sentence variety instead of commas, colons, or parentheses. Most human writers rarely use em dashes because they don't exist as a standard keyboard key.

**Replace em dashes as follows:**

| Instead of | Use |
|---|---|
| The results—which were surprising—showed... | The results, which were surprising, showed... |
| This approach—unlike traditional methods—allows... | This approach, unlike traditional methods, allows... |
| The study found—as expected—that... | The study found, as expected, that... |
| Communication skills—both written and verbal—are essential | Communication skills (both written and verbal) are essential |

**Rules:**
- Use commas for most parenthetical information
- Use colons to introduce explanations or lists
- Use parentheses for supplementary information
- Reserve em dashes for rare, deliberate emphasis only
- If you find yourself using more than one em dash per post, revise

## Phrases That Signal AI Writing

**Opening phrases — never use these:**
- "In today's fast-paced world..."
- "In today's digital age..."
- "In an era of..."
- "In the ever-evolving landscape of..."
- "In the realm of..."
- "It's important to note that..."
- "Let's delve into..."
- "Imagine a world where..."

**Transitional phrases — never use these:**
- "That being said..."
- "With that in mind..."
- "It's worth mentioning that..."
- "At its core..."
- "To put it simply..."
- "In essence..."
- "This begs the question..."

**Concluding phrases — never use these:**
- "In conclusion..."
- "To sum up..."
- "By [doing X], you can [achieve Y]..." (the generic formula)
- "In the final analysis..."
- "All things considered..."
- "At the end of the day..."

**Structural patterns — never use these:**
- "Whether you're a [X], [Y], or [Z]..." (listing three examples after "whether")
- "It's not just [X], it's also [Y]..."
- "Think of [X] as [elaborate metaphor]..."
- Starting sentences with "By" followed by a gerund: "By understanding X, you can Y..."

## Filler Words and Empty Intensifiers

These add nothing to meaning. Cut them or replace with something specific.

absolutely / actually / basically / certainly / clearly / definitely / essentially / extremely / fundamentally / incredibly / interestingly / naturally / obviously / quite / really / significantly / simply / surely / truly / ultimately / undoubtedly / very

## Academic-Specific AI Tells

| Avoid | Use Instead |
|---|---|
| shed light on | clarify, explain, reveal |
| pave the way for | enable, allow, make possible |
| a myriad of | many, numerous, various |
| a plethora of | many, numerous, several |
| paramount | very important, essential, critical |
| pertaining to | about, regarding, concerning |
| prior to | before |
| subsequent to | after |
| in light of | because of, given, considering |
| with respect to | about, regarding, for |
| in terms of | regarding, for, about |
| the fact that | that (or rewrite the sentence) |

## Self-Check Before Delivering

1. Read the post aloud. If any phrase sounds unnatural in speech, revise it.
2. Ask: "Would I say this in a conversation with a colleague?"
3. Check for repetitive sentence structures.
4. Scan for clusters of the flagged words above.
5. Verify sentence length varies — not all similar length.
6. Confirm every intensifier adds genuine meaning — if not, cut it.
